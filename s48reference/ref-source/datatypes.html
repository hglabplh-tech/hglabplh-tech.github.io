The basic datatypes like Integer , Boolean, Long, Float Double and the base functions

procedure

(not value) → boolean?

  value : any/c
This function takes a value as parameter and reverts the case of #t to #f

procedure

(eq? x y) → boolean?

  x : any/c
  y : any/c
This function takes a first value x as first parameter and a second value x and compares them for equality

procedure

(eqv? x y) → boolean?

  x : any/c
  y : any/c
This function takes a first value x as first parameter and a second value x and compares them for equality

procedure

(equal? x y) → boolean?

  x : any/c
  y : any/c
This function takes a first value x as first parameter and a second value x and compares them for equality

procedure

(max first rest ...) → number?

  first : number?
  rest : number?
This function takes a first value first as first parameter and a second or 0..n value(s) rest And here the highest value is returned

procedure

(min first rest ...) → number?

  first : number?
  rest : number?
This function takes a first value first as first parameter and a second or 0..n value(s) rest And here the lowest value is returned

procedure

(abs num) → number?

  num : number?
This function takes a number and returns the absolute which is than positive

  l : list?
takes the first element of a list from start of the list

procedure

  l : list?
takes the rest element(s) of a list all elements after the first element

3.1 PairsðŸ”—

procedure

(cons e1 e2) → pair?

  e1 : any/c
  e2 : any/c
constructs a pair out of two given elements to get a object which is a pair? as well as a list?

procedure

(pair? element) → pair?

  element : any/c
Tests if the element is a pair Examples:

(pair? (cons 5 6)) -> #t (pair? (list 5 6)) -> #t (pair? (list 5)) -> #f (pair? '(5 6)) -> #t (pair? '(5)) -> #f (pair? '()) -> #f (pair? 5) -> #f
