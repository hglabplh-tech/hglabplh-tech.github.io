procedure

(car l) → any/c

  l : list?
takes the first element of a list from start of the list

procedure

(cdr l) → list?

  l : list?
takes the rest element(s) of a list all elements after the first element

2.1 PairsðŸ”—

procedure

(cons e1 e2) → pair?

  e1 : any/c
  e2 : any/c
constructs a pair out of two given elements to get a object which is a pair? as well as a list?

procedure

(pair? element) → pair?

  element : any/c
Tests if the element is a pair Examples:

(pair? (cons 5 6)) -> #t (pair? (list 5 6)) -> #t (pair? (list 5)) -> #f (pair? '(5 6)) -> #t (pair? '(5)) -> #f (pair? '()) -> #f (pair? 5) -> #f

procedure

(member v l) → (or/c #f list?)

  v : any/c
  l : list?
Locates the first element of the list that is equal to v according to equal?. If such an element exists, the tail of lst starting with that element is returned. Otherwise, the result is #f.

procedure

(memq v l) → (or/c #f list?)

  v : any/c
  l : list?
Same as member with the difference that eq? is used instead of equal?

procedure

(memv v l) → (or/c #f list?)

  v : any/c
  l : list?
Same as member with the difference that eqv? is used instead of equal?
